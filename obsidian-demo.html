05

Код с подсветкой — прямо в заметке

Оберните код в тройные кавычки и укажите язык. Obsidian подсветит синтаксис. Поддерживается 50+ языков.

Python, JavaScript, TypeScript, Go, Rust, SQL...
Однострочный: `код`
Блок: три backtick + название языка
Копирование одной кнопкой
Python 📋 копировать 
# Персонаж книги как объект
class Character:
    def __init__(self, name: str):
        self.name = name
        self.connections = []
        self.secrets = []

    def connect(self, other, relation):
        self.connections.append({
            "who": other.name,
            "type": relation
        })

erik = Character("Эрик Ларсен")
sofia = Character("София Кейн")
erik.connect(sofia, "наставник")

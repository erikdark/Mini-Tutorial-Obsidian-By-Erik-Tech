Code with syntax highlighting — right inside your note

Wrap code in triple backticks and specify the language. Obsidian highlights the syntax. 50+ languages supported.

Python, JavaScript, TypeScript, Go, Rust, SQL and more
Inline code: `code`
Block: three backticks + language name
One-click copy button
Python 📋 copy 
# Book character as a data object
class Character:
    def __init__(self, name: str):
        self.name = name
        self.connections = []
        self.secrets = []

    def connect(self, other, relation):
        self.connections.append({
            "who": other.name,
            "type": relation
        })

erik = Character("Erik Larsen")
sofia = Character("Sofia Kane")
erik.connect(sofia, "mentor")
